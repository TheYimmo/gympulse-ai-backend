# Diccionario de datos — clean_gym_data.csv (SCRUM-17)

| Columna | Tipo | Descripción |
|---------|------|-------------|
| `country` | string | Nombre del país |
| `year` | int | Año calendario (2000–2026) |
| `region` | string | Región geográfica (Africa, Asia, etc.) |
| `gym_memberships` | float | Membresías activas en gimnasios |
| `fitness_participation_rate` | float | Tasa de participación fitness (0–1) |
| `total_health_club_revenue_usd` | float | Ingresos totales clubs USD |
| `number_of_gyms` | float | Cantidad de gimnasios |
| `gym_penetration_rate` | float | Penetración gym en población |
| `urban_population_percentage` | float | % población urbana |
| `obesity_rate` | float | Tasa de obesidad |
| `gdp_per_capita_usd` | float | PIB per cápita USD |
| `population_total` | float | Población total |
| `average_membership_cost_usd` | float | Costo promedio membresía USD |
| `insufficient_physical_activity_pct` | float | % actividad física insuficiente |

**Fuente:** [Kaggle dataset](https://www.kaggle.com/datasets/aryanmdev/world-gym-and-fitness-trends-20002026)
